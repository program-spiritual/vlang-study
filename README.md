# study-vlang
follow me to study v language

## 1. `v run`  command examples

### 1.1  run single file

```shell
v run hello.v
```

### 1.2 run in any directory

#### 1.2.1 run in directory

```shell
cd ANT_PATH && v run .
```
#### 1.2.2 run with parameters

```shell
v run . --yourparam some_other_stuff
```
#### 1.2.3 run and keep executable file

```shell
v -keepc run .
```
